# SQL Employee Analysis

## 📌 Overview
This project demonstrates SQL skills using an employee database.

## 🧱 Database Structure
- Employees table
- Departments table

## 🔍 Key SQL Concepts Used
- SELECT queries
- JOIN operations
- GROUP BY & Aggregations
- Subqueries

## 📊 Insights Generated
- Department-wise average salary
- Highest paid employee
- Employees earning above average
- Employee count per department

## 🛠 Tools Used
